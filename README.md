# Rust All Hands 2026

 The Rust All-Hands 2026 will take place as part of [RustWeek 2026](https://2026.rustweek.org/),
 which includes a conference and other activities.

> [!IMPORTANT]
> The Rust All-Hands is a *private event* for Rust team members and invited guests only.
> You are only invited if and only if you are a member of #rust-all-hands-2026-private on the Rust Zulip,
> or if you received explicit confirmation of your invitation by email from Mara.

## When and where

For the full schedule of RustWeek 2026, see the [RustWeek 2026 website](https://2026.rustweek.org/#week-schedule).

The All-Hands takes place on: **Thursday 21 May, Friday 22 May, and Saturday 23 May**.

The All-Hands will be held at the same venue as last year: [DUS, Sophialaan 1, Utrecht, Netherlands](https://2026.rustweek.org/dus/).
